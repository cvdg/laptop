# 2025-04-05

## Windows 11 VM

See: [How to Properly Install a Windows 11 Virtual Machine on KVM](https://sysguides.com/install-a-windows-11-virtual-machine-on-kvm)

## Download ISOs
```shell
$ sudo cp -v *.iso /var/lib/libvirt/images/
'virtio-win-0.1.266.iso' -> '/var/lib/libvirt/images/virtio-win-0.1.266.iso'
'Win11_24H2_EnglishInternational_x64.iso' -> '/var/lib/libvirt/images/Win11_24H2_EnglishInternational_x64.iso
```

## Create disk

```shell
$ sudo qemu-img create -f qcow2 /var/lib/libvirt/images/windows11.qcow2 128G
Formatting '/var/lib/libvirt/images/windows11.qcow2', fmt=qcow2 cluster_size=65536 extended_l2=off compression_type=zlib size=137438953472 lazy_refcounts=off refcount_bits=16
```
